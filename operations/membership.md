# Membership

## ` in ` Operation
Returns ` True ` if value is found in the sequence or array

**Example**
```py
>>> 1 in [1, 2, 3]
True

>>> 4 in [1, 2, 3]
False
```


## ` not in ` Operation
Returns ` True ` if value is not found in the sequence or array

**Example**
```py
>>> 1 not in [1, 2, 3]
False

>>> 4 not in [1, 2, 3]
True
```
