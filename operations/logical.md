# Logical

## ` and ` Operation
Returns ` True ` if both operands are ` True `

**Example**
```py
>>> True and True
True

>>> True and False
False

>>> False and False
False
```


## ` not ` Operation
Returns ` True ` if the operand is ` False ` in value

**Example**
```py
>>> not True
False

>>> not False
True
```


## ` or ` Operation
Returns ` True ` if either of the operands is ` True `

**Example**
```py
>>> True or True
True

>>> True or False
True

>>> False or False
False
```
