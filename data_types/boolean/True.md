## True
A boolean value that denotes something that is not ` False `. Can be created when a non-empty string or array, or non-zero integer, or of a conditional statement is passed to the ` bool() ` built-in function.

```py
>>> bool(1)
True

>>> bool("Python")
True

>>> bool([1, 2, 3])
True

>>> 1 in [1, 2, 3]
True

>>> not False
True
```
